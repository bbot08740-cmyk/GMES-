# Installation and Setup Guide

Follow these steps to import the bookmark collection into your browser.

### Step 1: Download the Bookmark File
1. Navigate to the download page: [Download](https://github.com/bbot08740-cmyk/GMES-/blob/main/Z.md).
2. Download the source file to your local device.

### Step 2: Open Bookmark Manager
* Press the following keyboard shortcut in your browser:
  * Windows / Linux: `Ctrl + Shift + O`
  * macOS: `Cmd + Option + B` (or use the browser menu to navigate to Bookmarks)

### Step 3: Import the File
1. Click the three vertical dots (menu icon) located in the top-right corner of the Bookmark Manager interface.
2. Select the **Import bookmarks** option from the dropdown menu.
3. Choose the downloaded file from your device to complete the process.
